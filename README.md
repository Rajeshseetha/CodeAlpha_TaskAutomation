Task Automation with Python Scripts

This project automates the process of extracting email addresses from a text file and saving them to a separate file.

Features:
- Reads data from a text file
- Extracts email addresses using Regular Expressions
- Saves extracted emails to a new file
- Reduces manual effort

Concepts Used:
- File Handling
- Regular Expressions (re)
- Loops
- String Processing

Technology Used:
- Python
